# @sveltopia/regions

Pass content from child pages to parent layouts in SvelteKit.

## Packages

- **[@sveltopia/regions](./packages/regions)** - The main library

## Apps

- **[site](./apps/site)** - Documentation and examples at [regions.sveltopia.dev](https://regions.sveltopia.dev)

## Development

```bash
pnpm install
pnpm dev        # Start all apps
pnpm build      # Build all packages
pnpm test       # Run tests
```
